Detailed Description

Classe abstraite, mère des classes StrategyAD et StrategyPR.

La classe CompteStrategy permet de diviser les opérations de crédit et de débit en deux classes : sur les comptes d'actifs/dépenses et sur les comptes de passifs/revenus.
